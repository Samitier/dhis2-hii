The project:

This project has the objective to offer a simple and intuitive way of reporting the state of health infrastructutes of a region, managing the information of the infrastructural state of the buildings and its inventory with periodical reports. This application will run on the information system https://www.dhis2.org and our main goal is to provide a complete integration with an existing DHIS2 infrastructure, and making a solution where the user doesn't have to configure anything to make it work.

This project is currently on development and a first v1.0 version is scheduled for the end of DecembeJanuary 2015.
